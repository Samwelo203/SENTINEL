How this system works

Every day, an automated job fetches environmental and health-system indicators for each county, engineers a 25-feature profile across five domains (environmental, WASH, mobility, health capacity, data quality), and runs it through disease-specific XGBoost models trained to estimate outbreak risk. SHAP analysis identifies which factors are driving each score, which the system turns into plain-language explanations and recommended actions. Scores reflect modeled risk based on conditions, not confirmed outbreaks.
